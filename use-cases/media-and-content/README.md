# Use Case: Media & Content

> 影片 / 廣告 / 電影。物理可控生成最大的商業 vertical。

## 三條商業路線

1. **TTV (text-to-video)** —— Sora / Veo / Kling / Wan
2. **Image-to-video** —— SVD / Cosmos-img2vid 線
3. **Controllable** —— ControlNet-video / trajectory / pose-cond

## 與其他 use-case 的差異

- Physics fidelity 容忍度高（觀眾不 care 物理嚴謹）
- 但「**物理感**」是觀眾感受 quality 的重要因素 —— 不嚴謹 ≠ 不在乎
- Controllability 在電影製作工程中重要性壓過 fidelity

## Dissection wishlist (2 篇)

- [ ] Kling / Hunyuan-Video / Wan SOTA 對比（中國商業線）
- [ ] ControlNet-video 在電影工程的整合
